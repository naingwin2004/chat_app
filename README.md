## Description  
This project was developed entirely using a mobile phone. While the UI/UX might not be as polished as applications built on a desktop.

### Note  
- This app was developed on a mobile phone, it might prioritize mobile usage.
- The UI was taken from online sources.


### Purpose  
This app serves as a learning project for MERN technologies, including:  
- React , Zustand  , DaisyUi
- MongoDB , Mongoose  
- Express  , JWT
- authentication & authorization methods
